INSTRUCCIONES DE SUBIDA A GITHUB / VERCEL

Este ZIP está preparado para Next.js y Vercel.

IMPORTANTE:
1. Descomprime el ZIP.
2. Sube a GitHub el CONTENIDO interno del ZIP, no la carpeta contenedora y no el archivo ZIP comprimido.
3. En la raíz del repositorio deben quedar visibles estos elementos:
   - package.json
   - app/
   - components/
   - public/
   - lib/
   - next.config.mjs
   - vercel.json

El error de Vercel:
"Couldn't find any `pages` or `app` directory"
aparece cuando Vercel no encuentra la carpeta app/ en la raíz del proyecto que está desplegando.

Cambios incluidos en esta versión:
- Se conserva el cambio del servicio Masaje Linfático con la imagen nueva.
- Se mantiene la imagen en tamaño original 109x105 px, sin expandir ni recortar.
- Se agregó el script vercel-build en package.json para Vercel.
- Se verificó que app/ y package.json estén en la raíz del ZIP.
